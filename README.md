# 🐉 PitayaBox-Java (火龙果纸箱 - Java版)

一款专为 Windows 玩家设计的轻量级腾讯 ACE 系统优化工具的 **Java 移植版本**。

本仓库 Fork 自原作者的 Rust 版本（Tauri），并使用 **Java 21 + JavaFX** 进行了完整的原生重构。在保留原版对恶霸后台进程强力限制的基础上，新增了更直观的 **UI 交互** 与 **系统级深度内存清理** 模块。

---

## ✨ 核心功能与亮点

### 🛡️ 进程主动限制 (同步原版)
* **CPU 亲和性锁定**：智能识别目标进程，强制将其绑定到系统的最后一个 CPU 核心，彻底解决后台进程抢占前台游戏核心资源的问题。
* **进程优先级压制**：一键将目标进程设为“空闲（Idle）”级别，游戏运行时绝不卡顿。
* **Windows 效率模式**：完美支持 Win11 专属 EcoQoS 调度，强制其进入低能耗状态。
* **I/O 读写与内存降权**：全方位限制磁盘访问与物理内存分配优先级。

### 🚀 系统内存深度清理 (Java版独占增强)
* **实时内存雷达**：集成 `OSHI` 跨平台监控，1秒级刷新系统内存真实占用曲线。
* **进程工作集清理**：调用 `EmptyWorkingSet` API，强制将闲置程序的物理内存挤压至虚拟内存。
* **系统缓存强行释放**：一键清空 Windows 庞大的文件预加载备用缓存（Standby List）。
* **自动化定时轮询**：支持自定义分钟级定时自动清理，让系统始终保持在最轻盈的状态。

### 🎨 现代感 UI 体验
* **极致流畅**：加入了丝滑的自定义平滑滚动算法（Smooth Scroll）。
* **一键暗黑/明亮主题**：无论是深夜修仙还是白天护眼，一键即可无缝切换完美适配的色彩体系。

---

## 🛠️ 技术栈与依赖
* **开发语言**：Java 21
* **UI 框架**：JavaFX 21
* **系统底层调用**：JNA (Java Native Access)
* **硬件信息获取**：OSHI (Operating System and Hardware Information)

## 🚀 快速开始

### 方法 1：直接运行已编译版本 (推荐)
1. 前往本仓库的 [Releases](你的仓库地址/releases) 页面下载最新版本的压缩包。
2. 解压后直接运行 `火龙果纸箱.exe` 即可（内置精简版 JRE，无需电脑额外安装 Java 环境）。

### 方法 2：源码编译
如果你想自行编译或进行二次开发：
```bash
# 1. 克隆本仓库
git clone [https://github.com/你的用户名/PitayaBox-Java.git](https://github.com/你的用户名/PitayaBox-Java.git)

# 2. 用 IntelliJ IDEA 打开
# 3. 使用 Maven 进行构建或打包
mvn clean package
